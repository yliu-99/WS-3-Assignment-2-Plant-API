# WS-3-Assignment-2-Plant-API

A full-stack digital plant shelf app built with React and Express. Track your houseplants with names, species, type, size, purchase date, and photos.

Built for **MDIA 4294 Web Scripting 3** at BCIT.

---

## Tech Stack

- **Front-end:** React, Vite, React Router
- **Back-end:** Express, MySQL2, Multer, CORS

---

## Features

- Browse all plants in a filterable grid
- Filter by plant type (Succulent, Tropical, Fern, etc.)
- View full details for a single plant
- Add a new plant with an image upload
- Edit an existing plant's details
- Delete a plant from the archive
